FarmaNet

FarmaNet es el backend de una aplicación para la gestión de farmacias, desarrollado con Java, Spring Boot y MongoDB. Proporciona todas las funcionalidades necesarias para manejar productos, clientes, ventas y carritos de compras a través de una API REST.

El backend se encarga de procesar la lógica de negocio, gestionar la persistencia de datos y asegurar que la información sea consistente y segura. Esta API puede ser consumida por el frontend, desarrollado en Angular, o por otras aplicaciones que necesiten interactuar con el sistema, permitiendo una comunicación fluida entre la interfaz de usuario y los servicios del servidor.
